# 项目说明

## 研究问题

扩散模型可以为小样本分类任务生成额外训练样本，但生成图像可能存在语义偏移、类别混淆和样本重复。本项目研究如何用 CLIP 从生成候选图像中筛选更有效的增强样本。

## 方法

CCDS 包含两部分：

1. **Class-Consistency Margin**：目标类别 CLIP 相似度减去最相似干扰类别相似度；
2. **Diversity Selection**：在高 margin 样本中基于 CLIP 图像特征聚类，选择更多样的生成样本。

## 主实验

- Real Only
- Traditional Augmentation
- Diffusion Random
- CLIP Top-K
- Margin Top-K
- CCDS
