# 运行指南

## 1. 激活虚拟环境

在 Git Bash / Claude Code shell 中：

```bash
cd D:/Astudy/clip_diffusion_fewshot_ccds
source .venv/Scripts/activate
```

如果是在 PowerShell：

```powershell
cd D:\Astudy\clip_diffusion_fewshot_ccds
.\.venv\Scripts\Activate.ps1
```

## 2. 创建 Flowers102 few-shot 划分

```bash
python scripts/make_splits.py --config configs/flowers20_5shot.yaml
```

## 3. 训练 baseline

```bash
python scripts/train_classifier.py --config configs/flowers20_5shot.yaml --method real_only --seed 0
python scripts/train_classifier.py --config configs/flowers20_5shot.yaml --method traditional_aug --seed 0
```

快速测试可以只跑 1 个 epoch：

```bash
python scripts/train_classifier.py --config configs/flowers20_5shot.yaml --method real_only --seed 0 --epochs 1
```

## 4. 本机小规模生成测试

你的 4060 Laptop 8GB 建议先跑：

```bash
python scripts/generate_candidates.py --config configs/flowers20_5shot.yaml --limit-per-class 2
```

完整实验再改为默认每类 80 张，建议放到 24GB 云端 GPU。

## 5. CLIP 打分

```bash
python scripts/score_candidates.py --config configs/flowers20_5shot.yaml
```

输出：

```text
results/clip_scores.csv
results/clip_image_embeddings.npz
figures/score_distributions/
```

## 6. 筛选候选图像

```bash
python scripts/select_candidates.py --config configs/flowers20_5shot.yaml --strategy all
```

输出：

```text
results/selected_random.csv
results/selected_clip_topk.csv
results/selected_margin_topk.csv
results/selected_ccds.csv
figures/selections/
```

## 7. 训练增强方法

```bash
python scripts/train_classifier.py --config configs/flowers20_5shot.yaml --method diffusion_random --seed 0
python scripts/train_classifier.py --config configs/flowers20_5shot.yaml --method clip_topk --seed 0
python scripts/train_classifier.py --config configs/flowers20_5shot.yaml --method margin_topk --seed 0
python scripts/train_classifier.py --config configs/flowers20_5shot.yaml --method ccds --seed 0
```

## 8. 绘图

```bash
python scripts/plot_results.py
```

## 9. 快速 smoke test

优先跑无扩散端到端 smoke test：

```bash
python scripts/run_quick_pipeline.py \
  --config configs/flowers2_1shot_smoke.yaml \
  --real-as-generated \
  --epochs 1 \
  --seed 0
```

这个命令会创建 2 类 1-shot 划分，训练 1 epoch baseline，用 held-out Flowers102 图片模拟 generated candidates，继续跑 CLIP 打分、筛选和 CCDS 增强训练。它用于检查代码链路，不作为正式实验结果。

如果只想测试 split + baseline：

```bash
python scripts/run_quick_pipeline.py --skip-generation
```

如果想连少量 Stable Diffusion 生成也测试：

```bash
python scripts/run_quick_pipeline.py
```

注意：第一次运行真实生成会下载 Flowers102、ResNet-50、Stable Diffusion 和 CLIP 权重，耗时较长。

## 10. 结果位置

按 `project_name` 隔离实验结果：

```text
results/<project_name>/generation_metadata.csv
results/<project_name>/clip_scores.csv
results/<project_name>/clip_image_embeddings.npz
results/<project_name>/selected/selected_*.csv
results/classifier/<project_name>/<method>/seed*/
results/classifier/all_results.csv
results/classifier/summary_by_method.csv
```

生成聚合表和图：

```bash
python scripts/plot_results.py --results results/classifier/all_results.csv
```
