## 0. 当前推荐入口：复查最佳 Pets20 结果

当前正式工作目录：

```bash
cd /root/clip_diffusion_fewshot_ccds/.claude/worktrees/anchored-ccds-results
```

轻量复查当前最佳结果（默认不重训、不重生成）：

单独运行：

```bash

PYTHONPATH=src /root/clip_diffusion_fewshot_ccds/.venv/bin/python scripts/summarize_best_pets20_results.py
```

## 9. 快速 smoke test

优先跑无扩散端到端 smoke test：

```bash
python scripts/run_quick_pipeline.py \
  --config configs/flowers2_1shot_smoke.yaml \
  --real-as-generated \
  --epochs 1 \
  --seed 0
```

这个命令会创建 2 类 1-shot 划分，训练 1 epoch baseline，用 held-out Flowers102 图片模拟 generated candidates，继续跑 CLIP 打分、筛选和 CCDS 增强训练。它用于检查代码链路，不作为正式实验结果。

如果只想测试 split + baseline：

```bash
python scripts/run_quick_pipeline.py --skip-generation
```

如果想连少量 Stable Diffusion 生成也测试：

```bash
python scripts/run_quick_pipeline.py
```

注意：第一次运行真实生成会下载 Flowers102、ResNet-50、Stable Diffusion 和 CLIP 权重，耗时较长。
